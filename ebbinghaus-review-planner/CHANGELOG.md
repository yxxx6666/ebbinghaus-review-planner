# Changelog

## v1.2.0 - Printable A4 Workbook Delivery

### Added

- 默认将学习系统制作成 A4 竖版 PDF 和可编辑 DOCX。
- 新增内容审核 -> A4 排版 -> DOCX 渲染 -> PDF 转换 -> PDF 渲染 -> 修正的强制闭环。
- 新增 `references/a4-print-workflow.md`，规定页边距、字号、色彩、表格、书写区和文件命名。
- 新增题目区与答案区分离规则，避免打印练习时直接看到答案。
- 新增封面、使用说明、练习区、复习日历、自评分、薄弱点和独立答案区结构。
- 新增 `scripts/check_a4_pdf.py`，检查 A4 页面尺寸、方向、可打开性和近空白页。
- 新增 DOCX 与 PDF 逐页视觉审核要求。

### Changed

- Markdown 改为内容底稿，不能再作为默认最终成品。
- 快速版、标准版和完整版都必须形成可打印手册；超长资料按分册处理。
- 打印版不再直接沿用聊天中的“问题 + 答案同表”结构。
- 交付前质量检查增加页面、打印、文件和答案编号审核。

## v1.1.0 - Precision and Adaptive Planning

### Changed

- 收窄自动触发：只有用户明确要求学习、记忆、复习、备考、培训或长期掌握时才使用。
- 增加快速版、标准版、完整版，控制知识点、题目和表格规模。
- 将“适当提前/延后”改为 0-4 分的确定性间隔算法。
- 增加最短 1 天、最长 90 天和连续高低分规则。
- 增加用户时区、实际学习日、未来开始日、同日合并、R0 冲突和每日预算规则。
- 增加考试截止处理，不再机械生成无关的考后冲刺任务。
- 调整 Notion 数据结构为纵向状态模型，减少横向复习日期字段。
- 强化 PDF 页码、PPT 编号、时间戳、章节、段落和关键原句定位。
- 重写黄金示例，展示真实短材料到 7 天任务的完整转换。

### Validation

- 升级本地校验器，检查 frontmatter、版本一致性、引用文件、临时文件和 UTF-8。

## v1.0.0 - Initial Release

- 首次发布知识地图、主动回忆题和间隔复习规划能力。
