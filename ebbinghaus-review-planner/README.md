# 艾宾浩斯复习规划大师

`ebbinghaus-review-planner` 把用户明确要学习、记忆、复习或长期掌握的资料，转成知识地图、主动回忆题、自适应复习日期、每日任务，以及经过逐页审核的 A4 可打印学习手册。

Current version: v1.2.0

## 适用场景

- 课程、考试和考证资料
- 书籍与论文学习
- 企业培训、产品知识、SOP 和销售话术
- 闪卡、Notion、Excel、Anki 或日历复习计划
- A4 学习手册、练习册和复习打卡表

## 不适用场景

只总结、翻译、润色、提取或转换文件时不应触发，除非用户同时明确要求学习或复习系统。

## v1.2.0 核心能力

- 默认交付 A4 PDF + 可编辑 DOCX
- 内容审核后再排版，排版后逐页渲染审核
- 题目区与答案区分离，保护主动回忆效果
- A4 竖版、打印安全边距、最小字号和宽表处理规则
- PDF 页面尺寸与近空白页自动预检
- 快速版、标准版、完整版三种输出规模
- 0-4 分自适应复习算法和 1-90 天间隔边界
- 学习日、考试日、时区、同日冲突和每日预算处理
- 原文页码、幻灯片、时间戳、章节或关键原句定位
- 长资料分批、分册处理和稳定编号
