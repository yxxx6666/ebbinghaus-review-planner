# 知识类型与主动回忆题型

## 1. 总原则

主动回忆题不是把所有知识点都变成问答卡。题型必须根据知识类型选择。

复习动作要可检测：能说出、能写出、能判断、能对比、能复述、能应用、能讲给别人听。

## 2. 类型映射表

| 知识类型 | 推荐题型 | 复习方式 | 示例检测标准 |
|---|---|---|---|
| 定义类 | 问答卡、填空卡 | 闭卷说出定义 | 能说出关键词，不改变原意 |
| 概念类 | 一句话解释卡 | 用自己的话解释 | 外行也能听懂 |
| 流程类 | 步骤复述卡 | 不看资料复述步骤 | 顺序正确，关键步骤不漏 |
| 步骤类 | 排序题、流程复述题 | 写出步骤并解释目的 | 能按顺序执行 |
| 公式类 | 公式默写卡 | 写出公式并解释变量 | 公式、单位、变量含义正确 |
| 数字类 | 填空卡、对比题 | 填出关键数字并说明意义 | 数字准确，知道使用场景 |
| 对比类 | 对比表 | 说出相同点与差异 | 至少列出 2-3 个关键差异 |
| 案例类 | 案例复述题 | 复述背景、冲突、结论 | 不混淆事实与观点 |
| 话术类 | 模拟对话卡 | 模拟真实对话 | 能自然说出关键话术 |
| 应用类 | 场景题、实操任务题 | 在真实场景中应用 | 能提出正确行动 |
| 理解类 | 讲给别人听题 | 讲给完全不懂的人听 | 能用类比或例子解释 |
| 判断类 | 判断题、纠错题 | 判断说法是否正确并说明理由 | 能指出错在哪里 |

## 3. 题型生成规则

### 问答卡

适合定义、概念、原因、原则。

格式：

| 字段 | 内容 |
|---|---|
| 问题 | X 是什么？ |
| 参考答案 | 根据原资料提炼，保留原意 |
| 检测标准 | 能说出核心关键词 |

### 填空卡

适合数字、公式、关键词、步骤名。

避免把太长的句子挖空。

### 判断卡

适合易错点、概念边界、误区。

必须提供判断理由，不能只给“对/错”。

### 选择题

适合考试模式。选项不能故意诡辩，错误选项应对应真实易错点。

### 场景题

适合工作应用、销售、SOP、产品知识、企业培训。

场景要贴近资料，不编造资料以外的事实。

### 流程复述题

适合 SOP、操作流程、工作流、方法论。

检测标准：顺序正确、动作完整、关键条件不漏。

### 对比题

适合概念边界、方案选择、优缺点。

优先用表格对比。

### 讲给别人听题

适合理解类、复杂概念、科普、书籍内容。

要求用户用普通人的语言解释。

### 实操任务题

适合工作应用和企业培训。

必须给出可执行动作和完成标准。
