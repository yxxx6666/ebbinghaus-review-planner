# 交付前质量检查

## 1. 是否应该触发

- [ ] 用户明确要求学习、记忆、复习、备考、培训或长期掌握。
- [ ] 不是单纯摘要、翻译、润色、提取或文件转换。

## 2. 内容忠实度

- [ ] 没有编造原文没有的事实、数字、案例、考点或答案。
- [ ] 不足信息标注“资料中未明确说明”。
- [ ] 原文依据尽量包含页码、幻灯片、时间戳、章节、段落或关键原句。
- [ ] 题目、答案、检测标准和原文依据相互一致。

## 3. 学习设计

- [ ] 知识点已拆成可回忆、可检测的小单元。
- [ ] A/B 级知识点至少有一道主动回忆题。
- [ ] 题型与知识类型匹配，不是全部问答卡。
- [ ] 答案包含必要关键词，且足够简短。
- [ ] 每道题有明确检测标准。
- [ ] 打印练习页不直接显示参考答案。

## 4. 输出规模

- [ ] 已选择快速版、标准版或完整版。
- [ ] 未超过默认数量上限；超长资料已分批或分册。
- [ ] 没有用超大表格假装完整。

## 5. 日期与负荷

- [ ] 优先使用用户提供的学习日、截止日和时区。
- [ ] 可计算日期均写成 YYYY-MM-DD。
- [ ] R0 与当天任务不足 2 小时时已合并。
- [ ] 同日重复阶段已合并。
- [ ] 有考试时未安排无关的考后冲刺任务。
- [ ] 每日任务未超预算；超出时已优先保留薄弱点和 A 级。

## 6. 自适应规则

- [ ] 使用 0-4 分评分。
- [ ] 低分项安排重新学习或缩短间隔。
- [ ] 连续低分标记薄弱点。
- [ ] 连续高分进入长期维护。
- [ ] 下一间隔限制在 1-90 天。

## 7. A4 文件结构

- [ ] 已生成 A4 竖版 PDF 和可编辑 DOCX，除非用户明确要求其他形式。
- [ ] 题目区和答案区分开。
- [ ] 有封面、使用说明、练习区、计划区、答案区和记录区。
- [ ] 表格列数适合 A4；过宽内容已改成纵向卡片或分表。
- [ ] 页码、标题层级、题号和知识点编号一致。

## 8. 逐页视觉审核

- [ ] DOCX 已渲染为页面图片并逐页查看。
- [ ] 最终 PDF 已渲染为页面图片并逐页查看。
- [ ] 每页尺寸为 A4，方向符合设计。
- [ ] 无裁切、重叠、乱码、黑块、断表、空白页或孤立标题。
- [ ] 正文字号不小于 9 磅，打印后可读。
- [ ] 表头在跨页表格中重复，行没有造成严重断裂。
- [ ] 黑白打印仍能区分标题、边框、勾选框和重点。
- [ ] PDF 页数、DOCX 页数和目录/页码关系合理。

## 9. 可执行性与交付

- [ ] 用户能看出今天做什么、何时做、做多久、如何判断完成。
- [ ] 没有把“再看一遍”作为唯一动作。
- [ ] 预计学习时长写成估算范围，未知长度时未编造精确分钟数。
- [ ] 最终文件名清楚，已交付 PDF 与 DOCX 链接。
- [ ] 未把内部渲染图、临时文件或校验日志当作最终附件。
