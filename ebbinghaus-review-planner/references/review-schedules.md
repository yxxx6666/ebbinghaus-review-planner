## 1. 基础原则

艾宾浩斯遗忘曲线只作为间隔复习启发。每次复习必须包含至少一个可检测动作：闭卷回忆、默写、做题、复述、讲解、对比或应用。

## 2. 四种基础节奏

### 2.1 标准版

| 阶段 | 时间 | 目标 | 推荐动作 |
|---|---|---|---|
| R0 | 学习后约 20 分钟 | 快速阻断遗忘 | 闭卷回忆 3-5 个核心点 |
| R1 | 当天晚上 | 第一次巩固 | 回答主动回忆题，标记低分项 |
| R2 | 第 1 天 | 稳固短期记忆 | 复述知识地图，做 A 级题 |
| R3 | 第 3 天 | 拉开间隔 | 闭卷解释重点概念与流程 |
| R4 | 第 7 天 | 周期复习 | 混合抽题，检查薄弱点 |
| R5 | 第 14 天 | 防止回落 | 场景应用或讲给别人听 |
| R6 | 第 30 天 | 长期记忆 | 综合测试，只保留薄弱项 |
| R7 | 第 60 天 | 长期巩固 | 低频抽查 |
| R8 | 第 90 天 | 长期维护 | 只复习 A 级和薄弱点 |

### 2.2 忙碌简化版

| 阶段 | 时间 | 推荐动作 |
|---|---|---|
| R0 | 当天 | 快速回忆核心知识点 |
| R1 | 第 1 天 | 做 A 级题 |
| R2 | 第 3 天 | 复述知识地图 |
| R3 | 第 7 天 | 检查薄弱点 |
| R4 | 第 14 天 | 混合复习 |
| R5 | 第 30 天 | 长期抽查 |

### 2.3 考试冲刺版

基础节点为学习后约 20 分钟、当天晚上、第 1、2、4、7、10 天和考前 1 天。只保留发生在考试日前的节点。

优先级：高频考点 > 易错点 > 错题 > A 级知识点 > B 级知识点。

### 2.4 兴趣轻量版

| 阶段 | 时间 | 推荐动作 |
|---|---|---|
| R0 | 当天 | 用自己的话复述主题 |
| R1 | 第 3 天 | 回忆关键概念 |
| R2 | 第 7 天 | 讲给别人听 |
| R3 | 第 14 天 | 写一段学习总结 |
| R4 | 第 30 天 | 抽查最有价值的知识点 |
| R5 | 第 60 天 | 判断是否长期保留 |

## 3. 自适应评分算法

### 3.1 评分

| 分数 | 判定 |
|---|---|
| 0 | 完全不会，无法开始回答 |
| 1 | 模糊，需要明显提示 |
| 2 | 能答出主要内容，但不稳定或有遗漏 |
| 3 | 能独立、准确、完整回答或讲清楚 |
| 4 | 能在新场景中应用、比较、举例或纠错 |

### 3.2 下一间隔

先计算：

`基础间隔 = max(1, 距上次复习的天数)`

| 分数 | 下一次间隔 |
|---|---|
| 0 | 1 天；当天补一次重新学习 |
| 1 | `max(1, round(基础间隔 × 0.5))` 天 |
| 2 | `max(1, round(基础间隔 × 1.2))` 天 |
| 3 | `min(90, max(2, round(基础间隔 × 2)))` 天 |
| 4 | `min(90, max(3, round(基础间隔 × 3)))` 天 |

文字反馈映射：不会=0，模糊=1，会=2，能讲出来=3，能应用=4。

### 3.3 连续表现

- 连续 2 次为 0 或 1：标记薄弱点，退回重新学习；生成“错误原因 + 正确解释 + 再测题”。
- 连续 3 次为 3 或 4：进入长期维护；按 30、60、90 天级别抽查。
- 最短 1 天，最长 90 天。
- 同一知识点按自己的表现调整，不用整份资料统一延后或提前。

## 4. 日期与冲突处理

1. 优先使用用户明确的学习日期。
2. 没有学习日期时使用当前日期。
3. 按用户时区计算。
4. 能计算时全部写 `YYYY-MM-DD`。
5. R0 与当天晚间任务相距不足 2 小时，合并为一次 10-20 分钟任务。
6. 同一天多个阶段合并，不重复列相同知识点。
7. 同日任务排序：薄弱点、A 级、B 级、C 级。
8. 学习开始日在未来时，不安排提前任务。
9. 无法获得当前日期时才使用 Day 0、Day 1 等相对日期。

## 5. 截止日与考试处理

1. 删除考试或截止日之后的冲刺节点。
2. 考前 1 天安排轻量复习，不塞入新知识。
3. 考试当天默认不安排高强度任务；如需安排，只做 5-10 分钟关键词回忆。
4. 用户要求长期掌握时，才能在考试后追加 30/60/90 天维护。
5. 时间不足时压缩知识范围，不把多个完整阶段挤到同一天。

## 6. 每日负荷控制

未提供预算时，默认每日最多 30 分钟。

超过预算时：

1. 保留薄弱点和 A 级知识点。
2. B 级拆分到下一可用日期。
3. C 级改为低频抽查或删除。
4. 单次任务尽量控制在 10-30 分钟。
5. 同一知识点一天内不要安排多次同型题；可用一次回忆加一次应用代替重复阅读。
