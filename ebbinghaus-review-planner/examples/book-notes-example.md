# 黄金示例：从短材料到可执行复习计划

## 用户输入

我今天学了下面这段内容，帮我做一个快速复习计划：

> 主动回忆不是重新阅读，而是在不看答案的情况下尝试说出或写出内容。检索越费力但仍能成功，记忆通常越牢。答错后要立即核对并纠正，避免重复错误。复习时应把注意力优先放在不会和模糊的内容上，而不是平均重复所有内容。
