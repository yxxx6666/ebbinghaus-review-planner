# 艾宾浩斯复习计划表

`ebbinghaus-review-planner` 是一个面向 ChatGPT / Codex 的学习与复习规划 Skill。它把书籍、PDF、课程资料和培训文档转成知识地图、零基础教学卡、主动回忆题、自适应复习计划，以及经过审核的 A4 学习手册。

当前版本：`1.3.0`

## v1.3.0 升级

- 完全不会模式：先教会，再测试。
- 零基础教学卡：大白话解释、正例、反例、拆解步骤、引导练习。
- 每日唯一任务：合并学习、复习、测试和应用，避免重复计算时间。
- 学习范围完整性检查：精读、略读、选读、暂不读。
- A/B 级知识点参与复习调度。
- 自适应记录：当前间隔、新间隔、下次日期。
- A4 PDF + DOCX 学习手册输出。

## 示例

本 Skill 可以把一本书变成：

- 知识地图
- 主动回忆题
- 复习节点
- 每日学习任务
- A4 打印手册

## 安装

```bash
npx skills add https://github.com/yxxx6666/ebbinghaus-review-planner --skill ebbinghaus-review-planner
```

## 使用

```text
$ebbinghaus-review-planner
我完全不会这份资料，请从零开始教我，并生成复习手册。
```

## 输出流程

资料解析 → 零基础教学 → 知识地图 → 主动回忆 → 复习计划 → A4排版 → 内容审核 → 页面审核 → 交付。
